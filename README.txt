# CloudSimExample

This project demonstrates a basic CloudSim simulation with VMs and Cloudlets.

## How to Run

- Import the project into your Eclipse IDE.
- Add required CloudSim library JAR files to the build path.
- Run the `CloudSimExample1` main class.

## Project Structure

- `CloudSimExample1.java`: main class with simulation code.
- Libraries: CloudSim jar files (make sure to include these in your IDE).

## Output

The program prints Cloudlet execution details with aligned columns.
