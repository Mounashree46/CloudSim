# CloudSim Project

This project demonstrates basic cloud simulation using the CloudSim toolkit in Java.

## 🛠 Features

- Creation of a Datacenter
- 4 Virtual Machines (VMs)
- 8 Cloudlets with different configurations
- Manual Cloudlet-to-VM binding
- Output displays execution details (start time, finish time, etc.)

## 💻 Technologies Used

- Java
- CloudSim 3.0.3
- Eclipse IDE (or any Java IDE)
- Git and GitHub

## 📁 Project Structure
cloudsimproject/
├── src/
│ └── cloudsimproject/
│ └── CloudSimExample1.java
├── lib/
│ └── (CloudSim JAR files)
├── README.md


## 🔧 Setup Instructions

1. **Install Java & Eclipse**
2. **Add CloudSim JARs** to your Eclipse project's build path.
3. **Paste the source code** into `CloudSimExample1.java`.
4. **Run the project** and observe the console output.

## 🚀 How to Run

- Compile and run `CloudSimExample1.java`
- Make sure all required CloudSim JARs are included
- Check the console for simulation results
